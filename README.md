# vue-router原理解析

> A Vue.js project

## Install

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```

## Usage

> main.js

```js
import router from './krouter'
```

> krouter.js

```js
import VueRouter from './kvue-router'
```

> App.vue

```vue
<template>
  <div id="app">
    <router-view/>
  </div>
</template>
```
